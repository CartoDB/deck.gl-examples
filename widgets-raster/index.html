CARTO Widgets for Raster sources

This example showcases how to build widgets (charts and filters) into raster visualizations built with CARTO + deck.gl, coming from your cloud data warehouse. This raster covers land use in the US with a 20m resolution.

The widgets are calculated on the client side and stay synchronized with the visualization, allowing all kinds of filtering and interactions. Learn more about widgets in CARTO.
