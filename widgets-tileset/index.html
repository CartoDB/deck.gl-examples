CARTO Widgets for Tileset sources

This example showcases how to build widgets (charts and filters) in a CARTO + deck.gl visualization that uses pre-generated tilesets. Our example tileset contains 2.7M lines representing water streams in the USA.

Widgets for tilesets are calculated client-side, and are synchronized with the map using spatialFilter. Learn more about building charts and widgets with CARTO.
