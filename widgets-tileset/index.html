Widgets for tilesets

This is an example of a couple scalable, performant Widgets using CARTO + deck.gl for a big USA river stream network.

These widgets aggregate data in different numerical categories, and they are synchronized with the map using spatialFilter. Learn more about building charts and widgets with CARTO.
